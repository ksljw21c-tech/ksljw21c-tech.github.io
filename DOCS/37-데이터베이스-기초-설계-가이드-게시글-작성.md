- 설계 프로세스: 요구사항 분석 → 개념적 설계 → 논리적 설계 → 물리적 설계

### SQL 기초
- DDL: CREATE, ALTER, DROP
- DML: INSERT, SELECT, UPDATE, DELETE
- JOIN: INNER JOIN, LEFT JOIN
- 인덱스와 성능 최적화
