### 바이브코딩 개념
- 안드레이 카르파티(OpenAI 공동 창업자)가 처음 사용한 용어
- AI와 자연어로 협업하는 새로운 코딩 방식
- "코드가 있다는 사실조차 잊어버리고 느낌에 맡기는 개발"
- 비개발자도 접근 가능한 프로그래밍 방식

### AI 도구들
- Cursor: VS Code 기반 AI 코딩 에디터
- Claude Code: Anthropic의 터미널 기반 코딩 도구
- Gemini CLI: Google의 Gemini AI 명령줄 도구
- Replit: 웹 기반 AI 코딩 플랫폼
- YouWare: 바이브코딩 전용 플랫폼

### 교육 프로그램
- 비개발자 대상 5주 커리큘럼
- 바이브코딩 부트캠프 (위니브)
- 혼자 공부하는 바이브코딩 (한빛미디어)
- 실리콘밸리 개발자 워크플로우

### 핵심 스킬
- 효과적인 프롬프트 작성
- 프로젝트 개요 문서(initialize.md) 작성
- 사용자 흐름(User Flow) 설계
- 반복적 개선 프로세스
