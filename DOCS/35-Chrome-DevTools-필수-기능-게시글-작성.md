## 수집된 자료 분석
tavily-mcp를 통해 다음과 같은 최신 정보를 수집했습니다:

### 기본 기능
- Elements 패널: HTML 구조 실시간 수정, 요소 검사
- Styles 패널: CSS 속성 실시간 편집, Computed 탭, 박스 모델 시각화
- Console 패널: JavaScript 실행, 에러 메시지 확인
- Layout 탭: CSS Grid, Flexbox 디버깅

### 디버깅 기능
- Sources 패널: 브레이크포인트, Watch 표현식, Call Stack
- 조건부 브레이크포인트, 콘솔 편의 명령어 ($0, $, $$, copy)
- Network 패널: HTTP 요청/응답 모니터링, Waterfall 차트
- Throttling 기능: 느린 네트워크 시뮬레이션

### 성능 분석 기능
- Performance 패널: CPU/메모리 분석, Long Tasks 식별
- Lighthouse 패널: Core Web Vitals, SEO 점수 측정
- Memory 패널: 메모리 누수 진단, 힙 스냅샷
- Coverage 패널: 미사용 코드 분석

### 고급 기능
- Device Mode: 반응형 디자인 테스트, 모바일 시뮬레이션
- Sensors 패널: GPS 위치, 가속도계 시뮬레이션
- Security 패널: HTTPS 인증서, 혼합 콘텐츠 진단
- Snippets: 재사용 가능한 코드 저장

### 확장 기능
- React Developer Tools
- Vue DevTools
- 프레임워크별 특화 디버깅 기능
