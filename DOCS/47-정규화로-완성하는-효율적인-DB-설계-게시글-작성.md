### 데이터베이스 정규화의 기본 개념
- 1NF (제1정규형): 원자성, 반복 그룹 제거
- 2NF (제2정규형): 부분적 함수 종속성 제거
- 3NF (제3정규형): 이행적 함수 종속성 제거
- BCNF (보이스-코드 정규형): 모든 결정자가 후보 키
- 4NF, 5NF: 고급 정규화 단계

### 정규화 적용 사례
- 전자상거래 시스템 정규화
- 학생-수업 등록 시스템
- 사원-부서 관리 시스템
- 함수 종속성 분석과 테이블 분리

### 실전 고려사항
- 역정규화 전략 (성능 vs 무결성 균형)
- 인덱스, 캐싱, 파티셔닝 최적화
- OLTP vs 데이터 웨어하우스 적용 차이
- 정규화 자동화 도구 활용
