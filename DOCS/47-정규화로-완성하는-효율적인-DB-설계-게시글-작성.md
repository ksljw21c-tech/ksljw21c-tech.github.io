### 데이터베이스 정규화의 기본 개념
- 1NF (제1정규형): 원자성, 반복 그룹 제거
- 2NF (제2정규형): 부분적 함수 종속성 제거
- 3NF (제3정규형): 이행적 함수 종속성 제거
- BCNF (보이스-코드 정규형): 모든 결정자가 후보 키

### 정규화 적용 사례
- 학생-수업 등록 시스템
- 전자상거래 주문 시스템
- 직원-부서 관리 시스템
- 함수 종속성 분석과 테이블 분리

### 실전 고려사항
- 역정규화 전략 (성능 vs 무결성 균형)
- 정규화 자동화 도구 (ERD 도구, SQL 분석 도구)
- 단계적 정규화 적용 방법
